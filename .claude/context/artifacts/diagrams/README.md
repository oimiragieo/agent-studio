## Viewing Mermaid Diagrams

These diagrams use [Mermaid](https://mermaid.js.org/) syntax embedded in Markdown.

### In VS Code

Install the "Markdown Preview Mermaid Support" extension.

### In GitHub

Mermaid diagrams render automatically in GitHub Markdown preview.

### Online

Use [Mermaid Live Editor](https://mermaid.live/) to paste and view diagrams.
