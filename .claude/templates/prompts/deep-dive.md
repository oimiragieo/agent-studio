# Deep Dive Prompt Template

## Purpose
Detailed analysis of specific code areas or features.

## Template

```
I need a deep dive into [AREA/FEATURE]. Please:

1. **Context**:
   - Explain the purpose and role of [AREA/FEATURE]
   - Identify where it fits in the overall system
   - Understand its dependencies and dependents

2. **Implementation Details**:
   - Analyze the code structure
   - Explain key algorithms and patterns
   - Identify design decisions and trade-offs
   - Understand error handling and edge cases

3. **Code Quality**:
   - Assess code quality and maintainability
   - Identify potential issues or improvements
   - Review test coverage
   - Check for security concerns

4. **Documentation**:
   - Review existing documentation
   - Identify documentation gaps
   - Suggest improvements

5. **Recommendations**:
   - Suggest improvements or optimizations
   - Identify refactoring opportunities
   - Recommend best practices

Please provide a comprehensive analysis with code examples and recommendations.
```

## Usage
- Feature analysis
- Code review preparation
- Refactoring planning
- Performance optimization

## Agent
- code-reviewer
- architect
- developer
- performance-engineer
