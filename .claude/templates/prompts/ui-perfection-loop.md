# UI Perfection Loop Prompt Template

## Purpose
Iterative UI improvement with visual perfection focus.

## Template

```
I need to perfect the UI for [COMPONENT/PAGE]. Please:

1. **Visual Analysis**:
   - Assess visual hierarchy and layout
   - Evaluate design consistency
   - Check enterprise polish level
   - Review glassmorphism (if applicable)

2. **Accessibility Audit**:
   - Check WCAG compliance
   - Verify color contrast
   - Test keyboard navigation
   - Review screen reader compatibility

3. **Implementation**:
   - Apply visual improvements
   - Fix accessibility issues
   - Ensure design system compliance
   - Optimize for mobile responsiveness

4. **Validation**:
   - Run visual regression tests
   - Verify accessibility improvements
   - Check cross-browser compatibility
   - Validate responsive design

5. **Iteration**:
   - Calculate quality score
   - Identify remaining issues
   - Plan next iteration improvements

Target score: [SCORE] (default: 95)
Max iterations: [N] (default: 5)
```

## Usage
- UI quality improvement
- Accessibility enhancement
- Design system compliance
- Visual polish

- accessibility-expert
