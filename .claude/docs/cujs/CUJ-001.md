# CUJ-001: First-Time Installation

## User Goal

Successfully install and configure LLM-RULES in a new project.

## Trigger

User copies `.claude/` folder to their project directory.

## Workflow

**Execution Mode**: `manual-setup`

### Step 1: Copy Configuration

- User copies `.claude/` folder to project root
- User copies `CLAUDE.md` to project root
- Verify structure matches expected layout

### Step 2: Validate Installation

- Run `pnpm install` (optional, for validation scripts)
- Run `pnpm validate` to check configuration
- Verify all required files exist

### Step 3: Enable Hooks (Recommended)

- Open Claude Code preferences
- Point hooks directory to `.claude/hooks`
- Enable security-pre-tool.sh (PreToolUse)
- Enable audit-post-tool.sh (PostToolUse)

### Step 4: Test Installation

- Try a simple command: `/review` or natural language prompt
- Verify agents are loading
- Check hooks are working (if enabled)

## Agents Used

- None (manual setup process)

## Skills Used

- None (setup phase)

## Expected Outputs

- `.claude/` directory in project root
- `CLAUDE.md` in project root
- Validated configuration
- Hooks enabled (optional)

## Success Criteria

| Criterion                            | Measurement                                       | Target                                                          |
| ------------------------------------ | ------------------------------------------------- | --------------------------------------------------------------- |
| Configuration files copied correctly | File existence validation in `.claude/` directory | All required files (agents, skills, workflows, schemas) present |
| Validation passes without errors     | `pnpm validate` exit code                         | Exit code 0 (no errors)                                         |
| Agents can be invoked                | Test simple command execution                     | Agent responds to `/review` or natural language prompt          |
| Hooks working (if enabled)           | Hook execution logs                               | PreToolUse and PostToolUse hooks execute successfully           |

## Example Prompts

```
# After installation, test with:
"Help me understand how to use this tool"
"/review"
```
