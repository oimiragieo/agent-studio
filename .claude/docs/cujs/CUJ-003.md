# CUJ-003: Cross-Platform Setup

## User Goal
Sync LLM-RULES configuration across Claude Code, Cursor, and Factory Droid.

## Trigger
- "Sync to Cursor"
- "Bridge context to Factory"
- "Set up cross-platform"

## Workflow

### Step 1: Identify Target Platform
- User specifies target: Cursor or Factory
- Check if target platform config exists

### Step 2: Context Bridge
- context-bridge skill exports current state
- Syncs agent configurations
- Syncs rule selections
- Syncs workflow state

### Step 3: Platform-Specific Setup
- For Cursor: Copy `.cursor/` configuration
- For Factory: Copy `.factory/` configuration
- Validate platform-specific files

### Step 4: Verification
- Test in target platform
- Verify agents work
- Check configuration sync

## Agents Used
- None (skill-based)

## Skills Used
- `context-bridge` - Cross-platform sync

## Expected Outputs
- Synced configuration files
- Platform-specific setup complete
- Cross-platform compatibility verified

## Success Criteria
- ✅ Configuration synced correctly
- ✅ Target platform configured
- ✅ Agents work in target platform
- ✅ Context preserved

## Example Prompts
```
"Sync this context to Cursor"
"Bridge to Factory Droid"
"Set up Cursor IDE configuration"
```
