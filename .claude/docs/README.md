# LLM-RULES Documentation

This directory contains user-facing documentation for deploying, using, and understanding LLM-RULES.

## Documentation Structure

### Setup Guides (`setup-guides/`)
Step-by-step guides for setting up LLM-RULES on different platforms:
- **[Claude Code Setup Guide](setup-guides/CLAUDE_SETUP_GUIDE.md)** - Configure LLM-RULES in Claude Code
- **[Cursor IDE Setup Guide](setup-guides/CURSOR_SETUP_GUIDE.md)** - Configure LLM-RULES in Cursor IDE
- **[Factory Droid Setup Guide](setup-guides/FACTORY_SETUP_GUIDE.md)** - Configure LLM-RULES in Factory Droid

### Deployment
- **[Enterprise Deployment Guide](ENTERPRISE_DEPLOYMENT.md)** - Deploy LLM-RULES in enterprise environments with security, scalability, and compliance considerations

### Features
- **[Enterprise Features Reference](ENTERPRISE_FEATURES.md)** - Quick reference for all enterprise-ready features including cost tracking, session management, guardrails, and testing

### How It Works
- **[Migration Guide](MIGRATION_GUIDE.md)** - Migrate existing projects to the optimized rule structure
- **[Rule Index System Migration](RULE_INDEX_MIGRATION.md)** - Understand and migrate to the dynamic rule index system
- **[Advanced Tool Use](ADVANCED_TOOL_USE.md)** - Guide to Anthropic's beta features for optimizing tool usage
- **[Context Optimization Guide](CONTEXT_OPTIMIZATION.md)** - Best practices for managing context usage and token efficiency
- **[UI Perfection Tools](UI_PERFECTION_TOOLS.md)** - Tool integration guide for UI perfection workflows
- **[Agent Generation Process](AGENT-GENERATION.md)** - How agents are structured and generated across platforms

## Quick Start

1. **Choose your platform** and follow the appropriate setup guide in `setup-guides/`
2. **Review features** in `ENTERPRISE_FEATURES.md` to understand available capabilities
3. **Deploy** using `ENTERPRISE_DEPLOYMENT.md` for production environments
4. **Optimize** using `CONTEXT_OPTIMIZATION.md` and `ADVANCED_TOOL_USE.md` for best performance
